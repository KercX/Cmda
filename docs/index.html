Overview

Cmda is a lightweight, extensible, and modern command console framework for Roblox experiences.

Inspired by evaera/Cmdr but rebuilt with clean architecture, better UI, and easier customization.

Made with ❤️ by KercX

Features

Fully Typed Arguments

Custom argument types with parsing and validation.

Client & Server Validation

Safe command execution with hooks.

Modern UI

Beautiful dark console with history and smooth UX.

Extensible

Easy to add commands, types, and hooks.

Getting Started

Clone the repository and use Rojo to sync the project.

git clone https://github.com/YOUR_USERNAME/Cmda.git

API Reference

Cmda:Start()

Initializes the framework on client and server.

Cmda:RegisterCommandsIn(container: Instance)

Registers all commands from a folder.

Cmda:RegisterTypesIn(container: Instance)

Registers custom argument types.

Cmda:RegisterHooksIn(container: Instance)

Registers hooks (e.g. BeforeRun for permissions).
