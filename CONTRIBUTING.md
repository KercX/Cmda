# Contributing to Cmda

Thank you for considering contributing to **Cmda**!  
We welcome bug reports, feature requests, and pull requests.

## How to Contribute

### 1. Reporting Bugs
- Open a new issue using the **Bug Report** template.
- Provide clear steps to reproduce the bug.
- Include Roblox Studio version, Luau version, and error messages.

### 2. Suggesting Features
- Open a new issue using the **Feature Request** template.
- Explain why the feature would be useful and how it fits into Cmda.

### 3. Submitting Pull Requests
1. Fork the repository.
2. Create a new branch for your changes (`git checkout -b feature/amazing-feature`).
3. Make your changes following the existing code style.
4. Run linting and formatting:
   ```bash
   selene src
   stylua --check src
